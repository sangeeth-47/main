Portfolio

Age Guessing

I built an AI-powered Age Guessing Web App that predicts a person’s age from their uploaded photo. The frontend is designed with HTML, CSS, and JavaScript, and the backend uses a TensorFlow model hosted on Hugging Face, integrated through a secure API.
